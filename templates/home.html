Welcome to the Boston Price Prediction API

This API allows you to predict house prices in Boston based on various features.

How to Use

Send a POST request to the /predict endpoint with the required features in JSON format.

Example Request

{
    "CRIM": 0.00632,
    "ZN": 18.0,
    "INDUS": 2.31,
    "CHAS": 0.0,
    "NOX": 0.538,
    "RM": 6.575,
    "AGE": 65.2,
    "DIS": 4.0902,
    "RAD": 1.0,
    "TAX": 296.0,
    "PTRATIO": 15.3,
    "B": 396.9,
    "LSTAT": 4.98
}

Example Response

{
    "prediction": 24.0
}
